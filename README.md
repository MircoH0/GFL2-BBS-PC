# 追放官方社区电脑版页面调整
![jsdeliver hits](https://data.jsdelivr.com/v1/stats/packages/gh/MircoH0/GFL2-BBS-PC/badge?type=hits&period=week&style=rounded)

### 功能:
 - 宽屏布局
 - 自定义配色（默认是暗色搭配）
 - 自动隐藏帖子内的回复输入框（适合只看帖为主的）
 - 查看大图界面添加设置此图为页面背景功能
 - 贴内回复显示的图片可缩放
 - 帖内回复以最新/最早显示
 - 将帖子标题添加到网页的标题前
 - 帖子内添加回到顶端按钮
 - 帖内右侧边栏移除
 - 增加游戏角色查询入口
 - 设置页面背景

以上功能均可单独开关，设置入口在页面的右上角
